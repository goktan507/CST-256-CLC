get /groupsrest
Returns all groups in the JSON formattized Rest API (getAllGroups)
Using this URL, it will return all the Groups created in the application in json format

Return type

array[GroupModel]

Example data

Content-Type: application/json
[ {
  "owner" : "Safa Bayraktar",
  "members" : [ "Safa Bayraktar", "Jacob Cauthren" ],
  "name" : "CST-256 Programmers",
  "description" : "A Programming group of CST-256 students",
  "id" : 1
}, {
  "owner" : "Safa Bayraktar",
  "members" : [ "Safa Bayraktar", "Jacob Cauthren" ],
  "name" : "CST-256 Programmers",
  "description" : "A Programming group of CST-256 students",
  "id" : 1
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Returned all groups successfully
